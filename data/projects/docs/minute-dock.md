# MinuteDock

MeetNote として考えていた議事録支援ツールを、より実務寄りのプロダクトとして整理し直したのが **MinuteDock** です。

## 背景

- 会議後の文字起こし整理、要点抽出、共有メモ作成に時間がかかっていた
- 録音から要約までを一つの流れで扱える形にしたかった
- 単なる文字起こしではなく、あとで見返しやすい「会議の成果物」に寄せたかった

## 取り組んだこと

- 音声データの取り込みから文字起こしまでの導線を整備
- AI を活用して要点整理や議事録のたたき台を生成
- 会議内容を後から探しやすい、読みやすい UI を意識して構成

## 工夫した点

- 旧構想の MeetNote をベースにしつつ、継続利用しやすい体験へ再設計した
- 録音、文字起こし、要約を分断せず、作業の流れとしてつなげた
- 個人利用だけでなく、チームでの共有を見据えた議事録 UX を意識した

## 今後

- 検索性の強化
- 外部ツールとの連携拡張
