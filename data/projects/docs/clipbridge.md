# ClipBridge

**ClipBridge** は、macOS と Windows 間で「今コピーした内容」をそのままつなぐことを目的にした、クリップボード同期アプリです。

## 背景

- 複数端末を行き来しながら作業していると、テキストやファイルを別の PC に渡すだけでも意外と手間がかかる
- メモアプリやチャットへ一時的に貼る運用では、共有のためだけの余分なステップが増えてしまう
- 履歴管理ではなく、直前にコピーした内容をすぐ他端末へ橋渡しする軽い体験を作りたかった

## 主な機能

- 最新のテキストクリップボードを macOS / Windows 間で同期
- Google Drive をバックエンドに使ったシンプルな同期構成
- OAuth 2.0 による Google アカウント連携
- CLI 常駐アプリとしての実行に加え、Tkinter ベースのデスクトップ UI も用意
- PyInstaller により macOS / Windows 向けアプリとしてビルド可能

## 工夫した点

- クリップボード履歴アプリではなく「最新 1 件の橋渡し」に用途を絞って、体験を明確にした
- 自分で送信したデータを再取得して無限ループしないよう、同期状態の管理を実装した
- 機密情報を扱う可能性を考え、ログに本文をそのまま出さない方針や同期 ON/OFF 設定を用意した
- ファイル同期も見据えて、テキスト同期と共存できる拡張しやすい構造にした
