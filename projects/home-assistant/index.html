Home Assistant

Home Assistant を中心に、スマートロック、赤外線家電、各種センサーを統合し、ローカル中心で運用できる自宅向けスマートホーム基盤を構築した。

課題と目的

家の中の機器を便利に操作したい一方で、機器ごとにアプリや操作方法が分かれていると管理が煩雑になる。また、クラウド依存の大きい構成では、外部サービスやネットワーク状況に左右されやすいという課題がある。

そこで、自宅内の機器やセンサーを Home Assistant に集約し、できる限りローカルで完結するスマートホーム基盤を構築した。目標は、単なる遠隔操作ではなく、鍵、家電、環境情報を一元管理し、日常的に使いやすく、今後も拡張しやすい運用基盤を作ることである。

主な機能と特徴

  • Sesame 5 の統合
    Sesame 5 を Home Assistant から扱う構成を検討・整備し、施錠・解錠状態の確認や遠隔操作をスマートホーム基盤に統合した。BLE を使うデバイスを Raspberry Pi 側で扱う前提で、ローカル構成を重視している。

  • 赤外線家電の制御
    ESP32 と ESPHome を用いて、テレビやエアコンなどの赤外線リモコン機器を Home Assistant から操作できる構成を作成した。赤外線送信距離の不足に対しては、IR LED の増設や外部電源、MOSFET 駆動も含めて検討している。

  • センサーデータの取得
    ESPHome デバイスを用いて、CO₂ センサーや温度センサーの値を取得し、Home Assistant 上で可視化できるようにした。環境データを監視するだけでなく、将来的な自動化条件として使える構成を意識している。

  • ダッシュボード運用
    Home Assistant 上で、スマホと PC の両方から使いやすいダッシュボード構成を検討した。リモコン画像を使った UI や、モバイル・デスクトップで表示を切り替える構成も試している。

  • 安全なリモートアクセスと保存先の分離
    Tailscale を用いた安全なリモート接続を視野に入れつつ、自宅外からも安定してアクセスできる構成を検討している。また、NAS との連携も見据え、データや関連ファイルを家の中の基盤に集約しやすい構成を目指している。

工夫した点・技術的な挑戦

このプロジェクトでは、異なる通信方式を持つ機器を一つの基盤にまとめる点が大きな課題だった。Sesame 5 は BLE、ESPHome デバイスは Wi-Fi、既存家電は赤外線で動作するため、それぞれを個別に扱うのではなく、Home Assistant を中心に役割を整理しながら統合した。

特に赤外線送信では、単純な構成では到達距離が不足しやすく、実用的な運用のために LED の個数、駆動回路、外部電源の必要性まで踏み込んで検討した。市販ハブを使えば簡単に済む部分でも、自作寄りの構成で仕組みを理解しながら進めている。

また、日常的に使う基盤にするためには、単にデバイスが動くだけでなく、どこからでも安全にアクセスできること、構成を将来的に拡張できることも重要だと考えた。そのため、リモート接続方法や補助サーバー、NAS の活用も含めて、家庭内のシステム全体として設計している点が特徴である。

構成

  • Home Assistant 本体
    Raspberry Pi 4 上で Home Assistant OS を動作させ、デバイス統合、自動化、ダッシュボード表示の中心として利用している。

  • 補助サーバー環境
    別の Raspberry Pi 4 上に Ubuntu 環境を構築し、Home Assistant 本体とは分けて周辺アプリや独自機能の実行基盤として利用している。

  • ESPHome デバイス
    ESP32 を用いて、赤外線送信、赤外線受信、CO₂・温度などのセンサー取得を行っている。

  • 統合対象機器
    Sesame 5、テレビ、エアコン、各種センサーを Home Assistant 上にまとめ、操作と状態確認を一元化している。

  • 周辺基盤
    NAS や補助サーバーとの連携も視野に入れ、スマートホームを単体機能ではなく、家全体の情報基盤として拡張しやすい構成を目指している。

今後やりたいこと

今後は、現在の操作・可視化中心の仕組みから、より自律的に動く生活基盤へ発展させたい。具体的には、CO₂ 濃度や室温、在宅状況などを条件にした自動制御を強化し、日常の操作をさらに減らしていきたい。

また、スマホでは素早く操作でき、PC では家全体を俯瞰できるように、ダッシュボードのレスポンシブ対応をさらに進めたい。加えて、独自ダッシュボードや 3D 表示との連携、NAS を活用したデータ蓄積や管理も進め、スマートホーム基盤をより拡張性の高い形に育てていきたい。
